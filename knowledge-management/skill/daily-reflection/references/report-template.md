```

  记忆总数:   {N} 条（user: {n} | repos: {n}）
  健康度:     {before}/100 → {after}/100
  问题汇总:   P0×{n} P1×{n} P2×{n}
  执行修复:
    - 删除记忆: {n} 条
    - 更新记忆: {n} 条
    - 合并记忆: {n} 条
    - 修正分类: {n} 条

  健康度:     {before}/100 → {after}/100

  执行操作:   {n} 项

```
