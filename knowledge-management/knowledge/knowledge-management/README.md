## 领域定义

**知识管理（Knowledge Management, KM）** 是一个跨学科领域，研究如何系统性地捕获、组织、存储、分享和应用知识，以实现个人或组织目标。

分为两大方向：
- **个人知识管理（PKM）**：个体层面的知识组织与学习方法
- **组织知识管理（OKM）**：企业/组织层面的知识创造与传递

## 核心概念地图

```
知识类型
├── 显性知识（Explicit）：可编码、可传递
└── 默会知识（Tacit）：基于经验、难以言说

知识转化（SECI螺旋）
├── 社会化（Socialization）：tacit → tacit（师徒传授）
├── 外化（Externalization）：tacit → explicit（文档化）
├── 组合（Combination）：explicit → explicit（系统整合）
└── 内化（Internalization）：explicit → tacit（学以致用）

个人KM流程
├── Capture → Organize → Distill → Express（CODE）
└── 卡片盒：原子化 → 链接 → 涌现
```
