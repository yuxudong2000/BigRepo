系统架构设计

MusicVision — Python + Canvas Web UI 完整架构

整体架构图
📥 输入层
音频文件 (MP3/WAV/FLAC/OGG)
↓ CLI: python visualize.py song.mp3
↓ Web: 拖拽上传 / 文件选择
📤 输出层
🎬 MP4 视频 / 🖼 GIF 动图
↑ moviepy + ffmpeg 合成
↑ Web 实时预览 + 一键下载
▼ 核心处理管线
🎵 AudioAnalyzer
librosa
• BPM / 节拍时间点
• 频谱 (STFT/Mel)
• 能量/响度
• 自动配色推断
🎨 VisualizerEngine
插件系统
• SpectrumBars
• WaveformFlow
• ParticleBurst
• CircularRadial
• + 自定义插件
📹 Renderer
双渲染路径
• Web: Canvas 实时
• CLI: 逐帧渲染
• moviepy 合成
• ffmpeg 编码
⚡ FastAPI Server POST /analyze | GET /stream/{job_id} | POST /export | GET /plugins
📁 项目结构
musicvision/
├── core/     # 音频分析 + 引擎核心
├── visualizers/ # 风格插件目录（每个风格一个文件）
├── web/      # FastAPI + 前端 HTML/JS/Canvas
├── cli.py     # 命令行入口
└── requirements.txt
