## 核心决策记录

### 产品定位
输入一段音频文件，系统自动分析音乐属性，生成酷炫吸引人的可视化视频。

### 可视化风格
确定四种内置风格，全部实现，并支持后续插件式扩展：
1. **频谱条形 Spectrum Bars** — 霓虹垂直柱，随 FFT 频谱跳动
2. **波形流动 Waveform Flow** — 渐变色平滑波形线
3. **粒子爆炸 Particle Burst** — 粒子在节拍时爆炸扩散
4. **环形放射 Circular Radial** — 圆心放射状频谱，随 BPM 旋转

### 输出格式
既要实时预览，也要能导出 MP4 + GIF（方案 D）。

### 参数控制
全自动：系统根据音乐分析结果自动选择颜色/速度/强度，无需手动调参。

### 运行方式
- **CLI**：`python cli.py song.mp3` 用于批量处理
- **Web UI**：浏览器拖入文件，实时预览 + 一键导出

### 技术方案
选择**方案 A：Python + Canvas Web UI**
- 音频分析：`librosa`（BPM、频谱、能量、音调）
- Web 服务：`FastAPI` + SSE 推送帧数据
- 前端渲染：Canvas 2D API
- 视频导出：`moviepy` + `ffmpeg`

### UI 布局
**左侧控制面板 + 右侧 Canvas 画布**
- 左侧：风格选择器 / 文件上传区 / 音乐信息（BPM、Key、能量条）/ 导出按钮
- 右侧：Canvas 实时渲染区域 + 底部播放控制条
