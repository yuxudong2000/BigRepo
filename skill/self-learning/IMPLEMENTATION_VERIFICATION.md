1. 显示降级警告和原因

5. 输出格式与 v1.0 兼容

**前提条件**:
- 并行模式启用
- 某个子 Agent 超时或失败

**预期行为**:
1. 其他分类正常完成
2. 失败的分类显示错误或跳过
3. 最终仍能生成归档摘要

## 🎓 技术亮点

### 1. 并行架构设计

借鉴 Hermes-agent `delegate_tool.py` 的并行批量模式：
- 7 个子 Agent 并发执行
- 独立上下文，互不干扰
- 容错机制完善

### 2. 原子写入保证

借鉴 Hermes-agent `memory_tool.py` 的原子写入模式：
- 临时文件 + os.replace
- fsync 强制落盘
- 错误自动恢复

### 3. 流式用户体验

借鉴 Hermes-agent `session_search_tool.py` 的流式输出：
- asyncio.as_completed 模式
- 实时进度反馈
- 感知速度优化
